# ТЗ на лендинг PALOMATIKA

## Общие параметры

- **Домен**: palomatika.ru
- **Тип**: Одностраничный лендинг с секциями
- **Адаптивность**: Desktop, Tablet, Mobile
- **Цветовая схема**: Тёмная тема

### Цвета

```css
:root {
    --bg-primary: #1A1A2E;      /* Основной фон */
    --bg-secondary: #16213E;    /* Вторичный фон */
    --bg-card: #0F3460;         /* Фон карточек */
    --accent: #E94560;          /* Акцент (красный/коралловый) */
    --accent-hover: #FF6B6B;    /* Акцент при наведении */
    --text-primary: #FFFFFF;    /* Основной текст */
    --text-secondary: #A0AEC0;  /* Вторичный текст */
    --success: #48BB78;         /* Зелёный */
    --error: #F56565;           /* Красный */
}
```

### Типографика

- **Заголовки**: Inter Bold / Montserrat Bold
- **Текст**: Inter Regular
- **Размеры**:
  - H1: 48px (desktop) / 32px (mobile)
  - H2: 36px (desktop) / 28px (mobile)
  - H3: 24px (desktop) / 20px (mobile)
  - Body: 16px / 18px

### Секция 3: Как это работает (интерактивное демо)

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│                 Не угадывай ответ — собери решение                     │
│                                                                         │
│     ┌─────────────────────────────────────────────────────────────┐    │
│     │                                                             │    │
│     │  ЗАДАЧА: В прямоугольном треугольнике катеты 3 и 4.        │    │
│     │          Найди гипотенузу.                                 │    │
│     │                                                             │    │
│     │  ────────────────────────────────────────────────────────  │    │
│     │                                                             │    │
│     │  Шаг 1: Запиши теорему Пифагора                            │    │
│     │                                                             │    │
│     │           c² = [____] + [____]                              │    │
│     │                                                             │    │
│     │  Блоки:  [a²]  [b²]  [a]  [b]  [a+b]                       │    │
│     │                                                             │    │
│     └─────────────────────────────────────────────────────────────┘    │
│                                                                         │
│     Ты собираешь решение по шагам. Система видит, где ты              │
│     ошибаешься, и подсказывает что повторить.                         │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Элементы:**
- Заголовок H2
- **Интерактивный пазл** (можно потрогать на лендинге!)
  - Drag-and-drop блоки
  - При правильном выборе — зелёная подсветка
  - При неправильном — красная + объяснение
- Подпись под демо

**Техническая реализация:**
- Alpine.js для интерактивности
- 1-2 шага демо-задачи
- Показать "ловушку" и объяснение
